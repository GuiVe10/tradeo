# Tradeo

PWA para vendedores de frutas y verduras. Gestiona oferta diaria, proveedores, clientes y pedidos; genera los mensajes de WhatsApp listos para copiar.

## Uso

Abrir `tradeo.html` en el navegador. Todo queda en `localStorage` bajo la clave `tradeo_data`. No hay backend.

Para instalar en el celular: abrir desde Safari o Chrome y usar "Agregar a pantalla de inicio".

## Stack

HTML + CSS + JavaScript vanilla en un único archivo. Sin dependencias.
